# Product Requirements Document (PRD)

## Project

Simple Todo

## Overview

Simple Todo adalah aplikasi Flutter untuk mencatat tugas harian secara offline. Fokus utama adalah aplikasi yang cepat, sederhana, dan mudah digunakan.

## Target User

- Pelajar
- Mahasiswa
- Freelancer
- Pekerja

## Goals

- Membuat aplikasi to-do sederhana
- Data tersimpan secara lokal
- UI modern menggunakan Material 3

## Features

### Core Features

- Add Task
- Edit Task
- Delete Task
- Mark Task Complete
- View Task List

### Additional Features

- Search Task
- Filter All
- Filter Completed
- Filter Active
- Dark Mode

## Data Model

Task

- id
- title
- isCompleted
- createdAt

## Scope

Included

- Offline only
- Local storage
- Material 3 UI

Excluded

- Login
- Cloud Sync
- Notification
- Category
- Calendar
