# Simple Todo

## Daily Workflow

1. Open project in VS Code
2. Connect Android phone
3. Run:
   ```bash
   flutter devices
   flutter run
   ```
4. Open Copilot Chat
5. Read:
   - CLAUDE.md
   - docs/PRD.md
   - docs/RULES.md
   - docs/TASKS.md
6. Continue the next unchecked task
7. Test:
   ```bash
   flutter analyze
   flutter run
   ```
8. Commit:
   ```bash
   git add .
   git commit -m "feat: ..."
   git push
   ```
